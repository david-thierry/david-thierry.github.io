Application Java

Application Java de location et de gestion de films.

L'Application Java

Ce projet de licence avait pour but de réaliser une application en java permettant de gérer des films en tant qu'administrateur, et de pouvoir en louer en tant que client. L'interface client nous permet différentes manipulations de l'application afin de regarder la liste des films disponibles, leur statut, leurs potentielles évaluations, etc...

Interface Administrateur

L'interface administrateur nous permet de gérer les films : Les supprimers, modifier leurs informations, en ajouter de nouveau au catalogue.
